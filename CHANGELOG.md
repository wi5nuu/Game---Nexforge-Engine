# Changelog

## [0.1.0] - 2026-06-18

### Added
- NexScript scripting language with lexer, parser, bytecode compiler, and stack-based VM
- Hot-reload system for live script iteration
- Archetype-based ECS with generational entities and columnar storage
- WebGPU rendering pipeline with PBR materials, cascaded shadow maps, and post-processing
- Physics engine on Rapier3d with custom BVH broad-phase
- Spatial 3D audio with HRTF and audio bus system
- AI system with behavior trees, NavMesh pathfinding, and utility AI
- Rollback netcode with GGPO-inspired deterministic simulation
- In-game developer console with command system
- Frame profiler with GPU timing and memory tracking
- Language Server Protocol implementation for NexScript
- First-person camera controls with WASD + mouse look
- 3D scene with animated objects and lighting
- 19 builtin functions in NexScript
- CI/CD with format check, clippy, test, build across 3 platforms
